# Bootstrap 5 Portfolio Website

## Project Overview
This project is a **responsive portfolio website** built using **Bootstrap 5**. The website contains three main pages:
- **Home Page**: Features a hero section and cards to showcase key features of the portfolio.
- **About Page**: Displays information about the services offered.
- **Contact Page**: Contains a contact form to allow users to get in touch.

The project is designed to be **responsive**, adapting seamlessly to different screen sizes (desktop, tablet, and mobile).

## Technologies Used
- **HTML5**: Used to structure the content of the web pages.
- **CSS3**: Used for additional styling and custom design elements.
- **Bootstrap 5**: Used for layout, components, and making the website responsive.
- **Git**: Used for version control.
- **GitHub**: Used for hosting the code repository.
- **Netlify/GitHub Pages**: Used for deployment.

## Features
- **Responsive Layout**: The site adjusts to different screen sizes (desktop, tablet, and mobile).
- **Navigation Bar**: Allows easy navigation between the pages (Home, About, Contact).
- **Feature Cards**: Displaying services or features in visually appealing cards.
- **Contact Form**: Provides a simple form for users to contact the website owner.
